This project is a React Native mobile application designed to manage and display information about various engine oils like Kixx, Havoline, and GP. Built using Tailwind CSS for responsive styling and React Navigation for seamless screen transitions, the app aims to provide a smooth user experience with a minimalistic design. The core functionality revolves around presenting engine oil details and recommendations in an intuitive and user-friendly way.

Key Features:
Oil Product Listing: Displays a list of oil brands (e.g., Kixx, Havoline, GP) with detailed descriptions.
Simple Navigation: Implemented with React Navigation, enabling users to switch between different sections easily.
Responsive Styling: Tailwind CSS is utilized for consistent, responsive, and modern UI design, optimized for various screen sizes.
Lightweight & Efficient: Designed to be fast and responsive, providing users with the essential information they need without unnecessary complexity.
